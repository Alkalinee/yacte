# Commands #

Here's a list of commands you can currently use.

| **Command**             | **Description**.                                   |
|:------------------------|:---------------------------------------------------|
| `:o [file]`             | Opens `[file]`                                     |
| `:s`                    | Saves content to file.                             |
| `:q`                    | Quits the editor, add `!` to force quit (`:q!`)    |
| `:x`                    | Saves the current file and then quits.             |
| `:r [find] [replace]`   | Replace `[find]` with `[replace]` (NYI).           |
| `:l`                    | List commands (NYI).                               |
